Here's my full review of the Google Antigravity website (antigravity.google):

Overview Google Antigravity is positioned as a next-generation agentic development platform — essentially a suite of AI-powered developer tools (desktop app, CLI, SDK, and IDE) built around the "agent-first era" concept. The site is clearly aimed at developers.

Design & Aesthetics ✅ The design is clean, modern, and polished — very much in line with Google's current brand language. The hero section features a subtle animated starfield/confetti particle background that gives it energy without being distracting. Typography is bold and confident (large, sans-serif display headings), and the color palette is minimal (white, black, with Google's signature colors used sparingly). The dark CTA section at the bottom adds a nice contrast. Overall, it's visually impressive and professional.

Navigation ✅ The nav is well-structured with clear categories: Product, Use Cases, Pricing, Blog, Resources — and a prominent "Download" CTA button. Dropdown menus are clean and descriptive, with sub-labels like "Explore our next generation products." Easy to navigate.

Hero Section ✅ Strong headline: "Experience liftoff with the next-gen agent platform." The animated cursor in the logo is a clever, on-brand touch. Two clear CTAs — "Download for MacOS" and "Explore use cases" — make the intended action obvious.

Product Section ✅ The four products (Antigravity 2.0, CLI, SDK, IDE) are each presented with concise descriptions and accompanying screenshots/visuals. The visuals are well-crafted and contextual (e.g., a dark terminal for CLI, a glowing orb for SDK). The "Explore Product" link is well-placed.

Use Cases Section ✅ The "Built for developers for the agent-first era" section is a nice touch, featuring video cards for Full Stack, Enterprise, and Frontend developers. The carousel format works well, though it could benefit from more use cases being visible at once on wider screens.

Pricing/Download Sections ✅ Simple messaging — free for individual developers, paid tier for organizations. The final dark CTA section with "Download for Apple Silicon" and "Download for Intel" options is clean and clear.

Blog Section ✅ The latest blogs section is well-organized with card layout, dates, and category tags. Content appears fresh (May 2026 posts), which builds credibility.

Footer ✅ Clean and functional. Links to Product, Docs, Changelog, Press, Releases, Blog, Pricing, and Use Cases. Google brand links (About Google, Privacy, Terms) are present. The giant "Antigravity" wordmark as a footer design element is a bold, memorable choice.

Areas for Improvement ⚠️

macOS only — The download buttons are exclusively for Apple Silicon and Intel Macs. There's no mention of Windows or Linux support, which could be a turn-off for a large chunk of developers. The site should at least acknowledge this limitation or provide a roadmap.
No pricing details visible on the homepage — While there's a "Pricing" nav item, the homepage only says "Available at no charge" for individuals and "Level up your entire team" for organizations, without any pricing specifics. This can feel vague.
Intro video placement — The "Play intro" button is there but not very prominent. A more visual teaser (like an auto-playing muted demo) could boost engagement.
Social proof is absent — No testimonials, user numbers, or logos from companies using the platform. For a developer tool, this could help build trust quickly.
Accessibility — Several icon-only elements (the scrolling icon strip) appear to lack descriptive labels, which could hurt screen reader accessibility.
Overall Rating: 8.5/10 It's a well-crafted, visually appealing product landing page that clearly communicates what Google Antigravity does and who it's for. With a few tweaks — platform clarity, social proof, and pricing transparency — it could be even stronger.

import { useEffect, useRef } from 'react'

export default function ParticleCanvas({ dark = false }) {
  const canvasRef = useRef(null)

  useEffect(() => {
    const canvas = canvasRef.current
    const ctx = canvas.getContext('2d')
    let animId
    let particles = []

    const resize = () => {
      canvas.width = canvas.offsetWidth
      canvas.height = canvas.offsetHeight
    }

    resize()
    window.addEventListener('resize', resize)

    const count = 120
    for (let i = 0; i < count; i++) {
      particles.push({
        x: Math.random() * canvas.width,
        y: Math.random() * canvas.height,
        size: Math.random() * 3 + 1,
        speedX: (Math.random() - 0.5) * 0.4,
        speedY: (Math.random() - 0.5) * 0.4,
        opacity: Math.random() * 0.6 + 0.2,
        hue: dark ? 210 + Math.random() * 40 : 330 + Math.random() * 30,
      })
    }

    const draw = () => {
      ctx.clearRect(0, 0, canvas.width, canvas.height)
      particles.forEach((p) => {
        p.x += p.speedX
        p.y += p.speedY
        if (p.x < 0 || p.x > canvas.width) p.speedX *= -1
        if (p.y < 0 || p.y > canvas.height) p.speedY *= -1

        ctx.save()
        ctx.globalAlpha = p.opacity
        // Draw a small elongated dash, like Antigravity
        ctx.translate(p.x, p.y)
        ctx.rotate(Math.atan2(p.speedY, p.speedX))
        ctx.fillStyle = dark
          ? `hsl(${p.hue}, 70%, 75%)`
          : `hsl(${p.hue}, 60%, 65%)`
        ctx.fillRect(-p.size * 1.5, -p.size * 0.4, p.size * 3, p.size * 0.8)
        ctx.restore()
      })
      animId = requestAnimationFrame(draw)
    }

    draw()
    return () => {
      cancelAnimationFrame(animId)
      window.removeEventListener('resize', resize)
    }
  }, [dark])

  return (
    <canvas
      ref={canvasRef}
      className="absolute inset-0 w-full h-full pointer-events-none"
    />
  )
}

export default function ParticleCanvas({ dark = false }) {
  const canvasRef = useRef(null)

  useEffect(() => {
    const canvas = canvasRef.current
    const ctx = canvas.getContext('2d')
    let animId
    let particles = []

    const resize = () => {
      canvas.width = canvas.offsetWidth
      canvas.height = canvas.offsetHeight
    }
    resize()
    window.addEventListener('resize', resize)

    for (let i = 0; i < 120; i++) {
      particles.push({
        x: Math.random() * canvas.width,
        y: Math.random() * canvas.height,
        size: Math.random() * 3 + 1,
        speedX: (Math.random() - 0.5) * 0.4,
        speedY: (Math.random() - 0.5) * 0.4,
        opacity: Math.random() * 0.6 + 0.2,
        hue: dark ? 210 + Math.random() * 40 : 330 + Math.random() * 30,
      })
    }

    const draw = () => {
      ctx.clearRect(0, 0, canvas.width, canvas.height)
      particles.forEach((p) => {
        p.x += p.speedX
        p.y += p.speedY
        if (p.x < 0 || p.x > canvas.width) p.speedX *= -1
        if (p.y < 0 || p.y > canvas.height) p.speedY *= -1
        ctx.save()
        ctx.globalAlpha = p.opacity
        ctx.translate(p.x, p.y)
        ctx.rotate(Math.atan2(p.speedY, p.speedX))
        ctx.fillStyle = dark ? `hsl(${p.hue},70%,75%)` : `hsl(${p.hue},60%,65%)`
        ctx.fillRect(-p.size * 1.5, -p.size * 0.4, p.size * 3, p.size * 0.8)
        ctx.restore()
      })
      animId = requestAnimationFrame(draw)
    }
